# Coaching Institute Management SaaS

A production-ready SaaS web application designed for coaching institutes to manage student applications and fee payments efficiently. The platform provides secure transaction handling, an admin dashboard, and a scalable architecture, and is actively used by real users in production.

---

## Overview

This application enables institutes to digitize their admission and fee workflows by offering a centralized system for application submissions, payment tracking, and transaction verification. It is built with a modern full-stack approach, focusing on performance, security, and maintainability.

---

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS, shadcn/ui  
- **Backend:** Express.js with RESTful APIs  
- **Database:** MongoDB with Prisma ORM  
- **Validation:** Zod for schema and data validation  

---

## Key Features

- Student application submission and management  
- Integrated payment tracking and transaction verification system  
- Role-based admin panel for institute management  
- Secure backend with validated APIs and structured data models  
- Scalable SaaS architecture with real-world production usage  

---

## Architecture Highlights

- Modular and maintainable codebase using TypeScript  
- Strong data integrity with Prisma and Zod  
- Clean, responsive UI built with reusable components  
- Designed for scalability and multi-institute usage
